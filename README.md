✨ 핵심 기능 (Key Features)

HRD ↔ IT 융합 지표 (Gradient Spectrum)

인재 육성(Human) 영역과 기술 개발(Tech) 영역의 내재된 밸런스를 그라데이션 슬라이더로 시각화하여 첫인상에서부터 '융합형 인재'임을 직관적으로 인지시킵니다.

인터랙티브 핀 타임라인 (Interactive Pin Timeline)

5대 핵심 커리어 성과를 시간 순으로 정렬하여 배치했습니다.

각 프로젝트 카드별로 HRD와 IT 기여 비중을 그라데이션 바로 표시하여 커리어의 성장 궤적을 투명하게 보여줍니다.

클릭형 상세 모달 (STAR Method Detail Modal)

타임라인의 각 핀을 클릭하면, 경험의 맥락을 분석할 수 있는 STAR(Situation, Task, Action, Result) 기법 기반의 구조화된 상세 설명이 매끄러운 애니메이션과 함께 나타납니다.

컨설턴트 맞춤형 질문 셀 (Consultant Q&A)

상세 모달 하단에 이 경험을 어떤 직무적 가치로 치환하면 좋을지에 대한 '구체적인 질문 피드백 카드'를 배치하여 양질의 컨설팅 피드백을 유도합니다.

성향 분석 섹션 (Soft Skills Overview)

'둥글둥글하고 유연한 성격'을 '이종 직군 간의 탁월한 번역가 및 공감 조율자'라는 역량 중심 언어로 재정의하여 협업 시너지를 시각화했습니다.
